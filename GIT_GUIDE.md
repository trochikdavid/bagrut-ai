# מדריך Git מהיר לשימוש יומיומי

כדי לעדכן את הקוד ב-GitHub, הרץ את הפקודות הבאות לפי הסדר:

1. **הוספת כל השינויים:**
   ```bash
   git add .
   ```

2. **שמירת השינויים עם הודעה:**
   ```bash
   git commit -m "כאן כותבים מה השתנה (למשל: עדכון עיצוב כפתורים)"
   ```

3. **שליחה ל-GitHub:**
   ```bash
   git push origin main
   ```

---

### פקודות שימושיות נוספות:
- `git status` - לראות מה המצב הנוכחי (איזה קבצים שונו).
- `git pull` - למשוך עדכונים מהענן למחשב (חשוב אם אתה עובד מכמה מחשבים).
